### Introduction
**Abstract**
- 機械学習，特に，深層学習は判断結果・根拠の解釈が難しい，つまり，なぜそのように判断したのかの根拠が示されない(**ブラックボックス**)という問題点がある．とりわけ，畳み込みニューラルネットワーク(CNN)は，画像から対象物を判別する時，画像のどこに着目しているのかを知りたい．
- CAMやGrad-CAMは，CNNが，どこに注目をして判断したのかを可視化できるツールである．
